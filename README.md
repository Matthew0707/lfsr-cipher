# LFSR Шифратор (Вариант 17)

**Потоковое шифрование на основе линейного регистра сдвига с обратной связью (LFSR)**

[![Открыть сайт](https://img.shields.io/badge/Открыть_Сайт-blue?style=for-the-badge)](https://Matthew0707.github.io/lfsr-cipher/)

## О проекте

Данный проект реализует потоковый шифр на основе LFSR с примитивным многочленом:

**P(x) = x^39 + x^4 + 1**

Разработан в рамках лабораторной работы №2 "Потоковое шифрование". Программа позволяет шифровать и дешифровать файлы любых форматов, используя генератор псевдослучайной последовательности на основе сдвигового регистра.

## Функциональность

- **Генерация ключевого потока** на основе LFSR с периодом 2^39 - 1 бит
- **Шифрование/дешифрование** файлов операцией XOR
- **Поддержка любых форматов** файлов (.txt, .png, .mp3, .jpg, .pdf и др.)
- **Визуализация** первых 10 байт исходного файла, ключа и результата
- **Отображение номера бита** при выделении
- **Cохранение** обработанных файлов

## Как использовать

1. **Введите начальное состояние** регистра (39 бит, только 0 и 1)
2. **Выберите файл** для обработки
3. Нажмите **"Запустить обработку"**
4. Программа выполнит шифрование (повторный запуск того же файла — дешифрование)
5. Файл сохранится автоматически с префиксом `processed_`

## Принцип работы

- Начальное состояние регистра выступает в роли секретного ключа
- LFSR генерирует псевдослучайную последовательность битов
- Каждый бит файла XOR-ится с соответствующим битом ключа
- Дешифрование выполняется повторным XOR с тем же ключом

## Особенности реализации

- Многочлен: **x^39 + x^4 + 1** (вариант 17)
- Отводы: биты 39 и 4
- Выходной бит: старший разряд регистра
- Интерфейс с фильтрацией ввода (только 0 и 1)
